Car finder


Welcome to the car finding application. Use the menu to navigate to different sections.


Project Description The goal of the project was to create a web application that allows users to browse and filter car data available on the otomoto.pl platform. The application uses Flask as the web framework, SQLite as the database, and Docker for container management. A parser built using BeautifulSoup collects data from the external site and stores it in the database.
